
- 모듈을 import하는 다른 방법
  - from 절을 활용해 특정 모듈을 미리 참조하고 어떤 요소를 import 할지 명시
```
from math import pi, sqrt


print(pi)

print(sqrt(4))
```
**두 코드중 명시적인것은 위 코드다**: **pi와 sqrt가 math의 코드인지 개발자가 직접 정의하고 만든 코드인지 알 수 없다.**
```
from math import * # *는 전체를 의미한다
```
